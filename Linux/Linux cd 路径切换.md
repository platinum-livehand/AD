## 进入指定目录

```shell
cd 目录名
```

## 返回上一级目录

```shell
cd ../
```

## 返回当前用户家目录

```shell
cd ~
```

## 在最近的两个目录之间切换

```shell
cd -
```

```shell
qy@localhost:~$ cd ./Data-Structure/
qy@localhost:~/Data-Structure$ cd -
/home/qy
qy@localhost:~$ cd -
/home/qy/Data-Structure
```
