## 进入指定目录

```cmd
cd 目录名
```

## 返回上一级目录

```cmd
cd ../
```

## 返回当前用户家目录

```cmd
$ cd ~
$ pwd
/home/qy
```

## 在临近的两个目录之间切换

```cmd
cd -
```

```cmd
$ cd ./Data-Structure/
$ cd -
/home/qy
$ cd -
/home/qy/Data-Structure
```

```cmd
cd /
```
